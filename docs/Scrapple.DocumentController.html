Scrapple.DocumentController

  • Summary
  • Functions

DocumentController handles serving structured markdown documents.

This controller will not list directory contents when a directory is specified. Rather, it will look for index.md or a scrapple.yml file that specified the document indexFile.

Methods in this controller will determine how to interpret relative links between markdown files and decorate the output accordingly.

NOT YET IMPLEMENTED:

  • scrapple.yml file
  • relative link decoration
